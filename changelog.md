# Changelog

All notable changes to Exobase are documented here, in [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
format. This file is kept in sync with the "What's New" list in Settings → Changelog
(`app/lib/api/features.ts`) — update both together.

Exobase is a fork of [bolt.diy](https://github.com/stackblitz-labs/bolt.diy); history from before the fork
lives in that project's own changelog and commit history, not here.

## [Unreleased]

### Changed
- Settings simplified from eleven tabs down to three: Profile, Changelog, Integrations.

## [0.1.0] - 2026-07-03

### Added
- Rebrand to Exobase on the structured.glass design system, dark-mode only.
- One-click GitHub connect via OAuth (Nango), replacing personal-access-token paste.
- Linear integration: connect an account and ask the AI to create issues directly from chat.
